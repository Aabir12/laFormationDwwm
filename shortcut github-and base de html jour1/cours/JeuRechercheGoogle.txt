1. Qu'est-ce que le HTML ?
HTML, ou HyperText Markup Language, est un langage de balisage utilisé pour structurer et présenter le contenu d'une page web. Il définit la structure des documents web en utilisant des balises pour organiser le texte, les images, les liens, les vidéos, et d'autres éléments interactifs.

2. Qu'est-ce que le CSS ?
CSS (Cascading Style Sheets) is a stylesheet language used to define the visual presentation and layout of web pages written in markup languages like HTML

4. Comment cloner un dépôt depuis GitHub ?
Accédez à la page du référentiel GitHub que vous souhaitez cloner.
Cliquez sur le bouton vert « Code ».
Copiez l'URL (choisissez HTTPS ou SSH selon votre configuration).
Ouvrez votre terminal (ou invite de commande).
Accédez au répertoire dans lequel vous souhaitez cloner le dépôt.
Exécutez la commande: git clone <repository-url>, remplaçant <repository-url> avec l'URL que vous avez copiée.
Appuyez sur Entrée et le référentiel sera cloné sur votre machine locale.

Quelle est la différence entre front-end et back-end en développement web ?
Les développeurs front-end s’occupent de l’interface de surface des sites Web, tandis que les développeurs back-end s’occupent de la communication des sites Web avec les bases de données et les serveurs

Qu'est-ce que PHP ?
PHP stands for PHP: Hypertext Preprocessor and is a widely-used, open-source scripting language primarily designed for server-side web development. It enables developers to create dynamic and interactive web pages by embedding PHP code within HTML. PHP is executed on the server, and the output is sent to the client as plain HTML, ensuring that the source code remains hidden from users.

Qu'est-ce que MongoDB ?
MongoDB is a NoSQL open-source database that stores data in flexible, JSON-like documents. Unlike traditional SQL databases that use tables and rows, MongoDB uses collections and documents, making it ideal for handling unstructured or semi-structured data
